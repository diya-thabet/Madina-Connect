#### Step 3: Run the Container
The command depends on your Operating System because Linux handles `host.docker.internal` differently.

**ðŸ‘‰ For Windows / Mac:**
```bash
docker run -d \
  -p 8001:8001 \
  --name planner-service \
  --env-file .env \
  madina-planner-llm

**ðŸ‘‰ For Linux (Ubuntu/Fedora):**
Linux docker doesn't support `host.docker.internal` by default. You must add the host gateway flag:
```bash
docker run -d \
  -p 8001:8001 \
  --add-host=host.docker.internal:host-gateway \
  --name planner-service \
  --env-file .env \
  madina-planner-llm
